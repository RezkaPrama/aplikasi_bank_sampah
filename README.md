Install :

    Edit file .env
    ketik “composer install” untuk install dependencies
    lalu “php artisan storage:link” untuk buat folder hasil upload
    Lalu “php artisan migrate” untuk buat db nya
    Lalu “php artisan db:seed”
    Untuk jalankan nya “php artisan serve”
